# roleplay.c

Roleplay.c is a multithreaded voice agent that specialize in small narrow AI models as the reasoning engine in order to get the best out of the limited resources.
Everything will be written in C in this project as the speed is paramount. Users can seamlessly swap the knowledgebase by swapping the language model.

## Features

- [x] Multithreaded server
- [x] Striving for < 50ms latency
- [x] GPU support
- [x] Embedded compatible

## Use cases

AI agent can act as a role play partner for the following use cases:

- [x] Sales conversations
- [x] Interviews
- [x] High stake negotiations
- [x] Date night conversations
